# MEMORY

## 주요 결정사항
* **구현 방식**: 방향 1 (프리렌더링 이미지 레이어 스위칭 방식)
  * **이유**: 개발 일정 단축 및 모바일/웹 브라우저에서의 시각적 완성도 극대화.
* **기능 범위**: 
  * 주얼리 디자인 커스텀 (금속 종류, 스톤 배열, 색상, 크기 선택)
  * 캐드 렌더 이미지 저장 및 상세 스펙 시트 (금속 종류, 중량, 보석 정보 등) 출력 기능 제공. (실제 3D CAD 파일 다운로드는 미지원)
  * 프로토타입 출력소 및 세공사 연계 기능 제공.
* **보석 영역 마스킹 (Gemstone Masking)**: 글로벌 CSS 필터를 이용한 금속 색상 변환 시 보석 영역이 오염되는 문제를 해결하기 위해 듀얼 이미지 레이어와 동적 SVG 마스크(mask)를 도입함.
  * **이유**: 금속 밴드 부분만 독립적으로 컬러 필터링을 적용하고, 스톤 영역은 원본 화질의 고대비 다이아몬드 광택과 보석 블렌드 컬러를 그대로 격리 보존하기 위함.
* **브랜드 무드**: 파인 주얼리 중심의 미니멀/고급스러움 (다크 모드 갤러리 컨셉)
